# Result Snapshots

These JSON files are validated outputs exported from the formal experiment runs in the working repository.

They are used to regenerate the paper figures and tables without requiring reviewers to rerun the full OpenFHE/GPU pipeline.

The current snapshot set includes:

- GPT-2 Small/Medium/Large 8-task utility results
- GPT-2 task-bucket latency, per-block breakdown, and sequence-length scalability
- GPT-2 security-level latency sensitivity for λ = 128, 192, and 256
- GPT-2 architecture-side and KD-side ablations
- BERT-base supplementary latency snapshots and MRPC utility comparison
